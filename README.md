# ZoomInBot 🤖🔍

A Telegram bot that zooms/enlarges images with 2x, 3x, and 4x magnification.

## Features
- 📸 Accepts images from users
- 🔍 3 zoom levels (2x, 3x, 4x)
- 🎨 High-quality LANCZOS resampling
- 💬 Interactive button interface
- 🚀 Deployed on Railway

## Environment Variables
- `TELEGRAM_BOT_TOKEN`: Your bot token from @BotFather
- `WEBHOOK_URL`: Your Railway app URL (e.g., https://zoom-in-bot.up.railway.app)

## Deployment
1. Fork/clone this repository
2. Create bot with @BotFather
3. Deploy on Railway
4. Set environment variables
5. Done!

## Commands
- `/start` - Show main menu
- `/help` - Show help
- `/cancel` - Cancel current operation
